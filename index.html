Consultas
El grupo de desarollo se encuentra siempre listo para responder sus dudas.

Para ello tenemos una linea fija para guiarlos en este nuevo proceso

Si desea realizar una sugerencia o reportar algún resultado inesperado, comuniquese de inmediato

Llama sin costos adicionales al: +50433223344

Login

Aquí deberá ingresar las credenciales que el equipo de desarrollo le ha asignado.

  • Conectar Sirve para iniciar sesión al sistema
  • Limpiar Util para quitar el texto ingreso
  • Salir Sirve para salir del sistema

El resto de botones brindan información

  • Recuperar Contraseña Sirve para contactarte con el equipo de desarollo
  • Tutorial Brinda información del sistema
  • Recordarme Sirve para salir guardar la contraseña de manera local

Inventario

Ventana donde podrá visualizar el listado de inventario actual

Los campos requeridos fueron señalados con el tipo de dato esperado. Ingrese solamente ese tipo de dato, no ingrese datos repetidos

  • Agregar Producto Sirve para iniciar sesión al sistema
  • Modificar Producto Seleccione el nombre del producto que desea modificar, posteriormente modifique los campos deseados, presione el botón
  • Eliminar Producto Seleccione el nombre del producto que desea eliminar, luego presione el botón, clic en aceptar
  • Limpiar Vaciará los datos ingresados
